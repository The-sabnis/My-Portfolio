# 🚀 Sarvesh Sabnis — Personal Portfolio

A modern, interactive personal portfolio website built with pure HTML, CSS, and Vanilla JavaScript.

Live at 👉 https://the-sabnis.github.io/portfolio

---

## 👨‍💻 About

This is my personal portfolio website showcasing my skills, experience, and projects as a **SQL Developer**, **Data Analyst**, and **Software Developer** based in Goa, India.

---

## ✨ Features

- ⚡ Pure HTML, CSS & Vanilla JavaScript — no frameworks, no dependencies
- 🎨 Dark theme with Electric Violet, Cyan & Amber color palette
- 🖱️ Custom 3-layer cursor — dot, ring & trail with hover/click states
- 💫 Particle network animation in hero section
- ⌨️ Typing animation cycling through roles
- 📜 Stepper accordion timeline for experience section
- 🃏 3D tilt effect on project cards
- 🔢 Animated counters for stats
- 📱 Fully responsive — mobile, tablet & desktop
- 🔄 Scroll-based reveal animations
- 📊 Scroll progress bar
